# Decision Log

## Purpose

Track important Karma Infinity LLC decisions in a clear and reviewable way.

## Decision template

```markdown
## YYYY-MM-DD — Decision title

### Status

Proposed | Accepted | Superseded | Rejected

### Context

What situation led to this decision?

### Decision

What was decided?

### Rationale

Why was this path chosen?

### Review notes

Legal, security, privacy, accessibility, and mission considerations.

### Trade-offs

What are we accepting or delaying?

### Follow-up actions

- [ ] Action 1
- [ ] Action 2
```

## Initial decision

## 2026-05-01 — Create the organization command center

### Status

Accepted

### Context

Karma Infinity LLC needed a professional GitHub foundation before creating operational and product repositories.

### Decision

Create a public `.github` repository for organization-wide community files, governance docs, templates, and profile content.

### Rationale

This gives future repositories a consistent foundation.

### Review notes

Keep legal status accurate: Karma Infinity LLC is a New Jersey limited liability company.

### Trade-offs

This repository is governance infrastructure, not product software.

### Follow-up actions

- [ ] Create standard labels
- [ ] Create `karma-command-center`
- [ ] Use `.github` defaults for future repositories
